THPS-THPG any% leaderboard

Why these specific categories?

These categories are meant to be like if you booted
up a game for the first time ever but for some reason
you had insane skill and knowledge already.
How quick could you get to the credits roughly in a
way that matches the intended way by Neversoft.
Basically the default any% catagories.

How are points calculated?

Using the same formula as thps.run's Packle Points.

Packle Points =
floor((0.008 * e^(4.8284*(fastest_time/time)))*1000)
with 1st place always getting 1000 points.
